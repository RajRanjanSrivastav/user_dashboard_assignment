# 🚀 Unstop Assignment - Angular User Dashboard

A modern Angular 17+ standalone dashboard application built with RxJS, Chart.js, and Tailwind CSS. It demonstrates real-world frontend engineering concepts like state management, lazy loading, and performance optimization.

---

## ✨ Features

- 👥 Dynamic User Management (Add / View users)
- 📊 Role Distribution Pie Chart (Chart.js)
- 🔍 Search + Filtering functionality
- 📄 Pagination for scalable user list
- ⚡ Lazy-loaded modal form for adding users
- 🧠 Reactive state management using RxJS BehaviorSubject
- 💾 Session storage persistence (data survives refresh)
- ⚡ Performance optimization using `@defer`
- 🎨 Clean Material-inspired UI using Tailwind CSS
- 🧩 Fully Standalone Angular Architecture (No modules)

---

## 🛠️ Tech Stack

- Angular 17+
- RxJS
- Chart.js
- Tailwind CSS
- TypeScript
